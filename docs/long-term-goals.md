## 1. 五年职业愿景

### 1.1 愿景描述

**5年后（28岁）的理想状态**：

```
职业定位：高级开发者 / 资深工程师 / 技术专家
          或
          独立开发者 / 创业者

技术能力：某一领域专家级别
          （图形学/网络/架构）

收入水平：年薪50-80万+
          或
          主业+副业综合收入更高

生活状态：工作与生活平衡
          有明确的人生方向
```

### 1.2 职业路径选择

**路径 A：技术专家路线**

```
初级 → 中级 → 高级 → 资深 → 专家
                            │
                      5年后位置
                      月薪40K-60K
                      年薪50-80万
```

**路径 B：技术管理路线**

```
初级 → 中级 → 高级 → Team Leader → 技术总监
                                  │
                            5年后位置
                            月薪50K+
                            年薪70万+
```

**路径 C：独立/创业路线**

```
初级 → 中级 → 高级 → 独立开发者/创业
                        │
                  5年后位置
                  收入不确定
                  自由度高
```

### 1.3 选择决策点

| 时间 | 决策点 | 考虑因素 |
|------|--------|---------|
| 第3年 | 专家 vs 管理 | 兴趣、能力、机会 |
| 第4年 | 继续 vs 创业 | 经济基础、风险承受 |
| 第5年 | 调整方向 | 实际发展情况 |

## 2. 财务目标规划

### 2.1 收入增长预测

| 年份 | 年龄 | 主业收入 | 副业收入 | 总收入 |
|------|------|---------|---------|--------|
| 第1年 | 23-24 | 15万 | 0 | 15万 |
| 第2年 | 24-25 | 25万 | 2万 | 27万 |
| 第3年 | 25-26 | 35万 | 5万 | 40万 |
| 第4年 | 26-27 | 45万 | 8万 | 53万 |
| 第5年 | 27-28 | 55万 | 12万 | 67万 |

### 2.2 财务目标

**存款目标**：

| 时间 | 目标存款 | 用途 |
|------|---------|------|
| 第3年 | 30-50万 | 应急储备 |
| 第5年 | 80-100万 | 首付/创业资金 |

**投资规划**：

```
资产配置建议
│
├── 活期/货币基金：10-20%
│   └── 应急资金
│
├── 稳健理财：40-50%
│   └── 债券、定期
│
└── 进取投资：30-40%
    └── 基金、股票
```

### 2.3 收入结构优化

```
理想收入结构（5年后）
│
├── 主业：60-70%
│   └── 稳定现金流
│
├── 副业：20-25%
│   └── 工具销售、咨询
│
└── 投资：10-15%
    └── 被动收入
```

## 3. 人生重大决策规划

### 3.1 决策时间线

```
时间线上的重大决策
│
├── 24-25岁（第2年）
│   ├── 职业方向确认
│   └── 恋爱关系发展
│
├── 25-26岁（第3年）
│   ├── 跳槽/晋升决策
│   └── 关系深化
│
├── 26-27岁（第4年）
│   ├── 城市选择
│   └── 可能的婚姻
│
└── 27-28岁（第5年）
    ├── 职业路径确认
    └── 家庭规划
```

### 3.2 城市选择

**留在上海 vs 其他城市**：

| 因素 | 上海 | 二线城市 |
|------|------|---------|
| 薪资 | 高 | 中等 |
| 生活成本 | 高 | 低 |
| 发展机会 | 多 | 较少 |
| 生活质量 | 中 | 高 |

**建议**：
- 前3年留在一线发展
- 根据情况评估是否迁移

### 3.3 创业 vs 打工

| 选择 | 时机 | 准备 |
|------|------|------|
| 继续打工 | 稳定优先 | 提升能力 |
| 内部创业 | 有机会时 | 项目积累 |
| 独立创业 | 有资金和想法 | 存款50万+ |

## 5. 副业转主业可能性

### 5.1 评估标准

**副业转主业的条件**：

| 条件 | 标准 |
|------|------|
| 收入稳定性 | 副业收入>主业50% |
| 增长潜力 | 有持续增长空间 |
| 风险可控 | 有足够储备金 |
| 热爱程度 | 愿意全职投入 |

### 5.2 可能的副业方向

| 方向 | 潜力 | 转主业可行性 |
|------|------|-------------|
| 工具开发 | 中 | 中等 |
| 独立游戏 | 高（但风险大） | 高 |
| 技术咨询 | 中 | 中等 |
| 教育培训 | 中 | 中等 |

### 5.3 决策建议

```
副业转主业决策流程
│
├── 评估现状
│   ├── 副业收入稳定性
│   ├── 市场增长潜力
│   └── 个人能力匹配
│
├── 风险评估
│   ├── 经济风险
│   ├── 机会成本
│   └── 失败后果
│
└── 做出决策
    ├── 时机合适 → 转型
    └── 时机不成熟 → 继续积累
```
